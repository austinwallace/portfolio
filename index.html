Skills/My Projects

StepSequencer

StepSequencer is a web application which allows the user to program a drum beat and the application will loop the drum beat back to them. The application allows musicians to create backing tracks for practice or live use. This was a solo project done during my time at the Digital Crafts coding bootcamp. StepSequencer is a front-end based project using React, JavaScript, HTML, and CSS.

Reddit Clone

Reddit Clone is a recreation of the popular site Reddit.com that allows users to make forum posts, upvote and downvote, and comment on each post. The application is front-end based using HTML, JavaScript, and CSS, and utilizes our own API.
